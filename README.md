# CSC1109 Data at Speed & Scale - Assignment 1

This repository contains my assignment for CSC1109 - Data at Speed and Scale.

This assignment uses Pig and Hive in the Hadoop ecosystem to clean and analyze University Rankings datasets from CWUR and Times Higher Education.

The steps taken to complete this assignment:

- Data was uploaded to HDFS and prepared for processing
- Datasets were cleaned using Pig scripts
- Simple queries were executed in both Pig and Hive
- Complex queries were performed using Hive
- Results were analyzed and documented

## Repository Contents

- **Final PDF** - Complete assignment documentation and analysis
- **Pig scripts** - Data cleaning and processing scripts
- **Hive queries** - Query files for data analysis
- **Screenshots** - Evidence of query outputs and results
- **Datasets** - Raw and processed university rankings data

## Technologies Used

- Apache Hadoop
- Apache Pig
- Apache Hive
- HDFS
